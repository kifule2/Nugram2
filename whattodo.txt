linkify
Here's how to make links clickable in your TikTok-style feed. You need to add a function that automatically detects URLs in post content and converts them to clickable links.

🔧 Fix: Make Links Clickable in Feed

1. Add a template filter to detect and convert URLs

Create a new file social/templatetags/social_filters.py:

```python
from django import template
from django.utils.safestring import mark_safe
import re

register = template.Library()

@register.filter
def linkify(text):
    """
    Convert URLs in text to clickable HTML links
    """
    if not text:
        return text
    
    # URL regex pattern
    url_pattern = re.compile(
        r'(https?://[^\s]+)|(www\.[^\s]+)|([a-zA-Z0-9.-]+\.[a-zA-Z]{2,}(?:/[^\s]*)?)'
    )
    
    def replace_url(match):
        url = match.group(0)
        # Add https:// if missing for www or domain-only links
        if url.startswith('www.'):
            full_url = 'https://' + url
        elif not url.startswith('http'):
            full_url = 'https://' + url
        else:
            full_url = url
        
        # Truncate long URLs for display
        display_url = url if len(url) <= 50 else url[:40] + '...'
        
        return f'<a href="{full_url}" target="_blank" rel="noopener noreferrer" style="color: #3b82f6; text-decoration: none; border-bottom: 1px solid rgba(59,130,246,0.3);">{display_url}</a>'
    
    result = url_pattern.sub(replace_url, text)
    return mark_safe(result)
```

2. Register the template tag

Make sure social/templatetags/__init__.py exists (can be empty).

3. Update your feed.html

First, load the custom filter at the top of your feed.html (right after {% extends "base.html" %}):

```html
{% extends "base.html" %}
{% load static %}
{% load social_filters %}
```

Then, wherever you display post content, wrap it with the linkify filter.

For text posts (around line where you show text content):

```html
{% if post.post_type == 'text' %}
    <div class="text-post-container 
        {% if post.background_template and post.background_template.css_class %}{{ post.background_template.css_class }}{% else %}gradient-sunset{% endif %}">
        <div class="text-post-content" id="text-content-{{ post.id }}">
            {% with content=post.content|linebreaksbr|linkify %}
                {% if post.content|length > 300 %}
                    <span class="short-text">{{ content|truncatechars_html:300 }}</span>
                    <span class="full-text" style="display: none;">{{ content }}</span>
                    <button class="see-more-btn" onclick="toggleTextContent({{ post.id }}, event)">See more</button>
                {% else %}
                    {{ content }}
                {% endif %}
            {% endwith %}
        </div>
    </div>
{% endif %}
```

For captions in media posts (in the post-info section):

```html
<div class="post-info">
    <a href="{% url 'users:user_profile' post.user.username %}" class="username">
        @{{ post.user.username }}
        {% if post.user.userprofile.verified %}<i class="fas fa-check-circle verified"></i>{% endif %}
    </a>
    {% if post.content and post.post_type != 'text' %}
    <div class="caption">{{ post.content|linkify|truncatechars_html:120 }}</div>
    {% endif %}
    <div class="audio">
        <i class="fas fa-music"></i>
        {{ post.song_name|default:"Original Audio" }}
    </div>
</div>
```

4. Add this CSS to make links look good

In your feed.html <style> block, add:

```css
/* Link styles */
.text-post-content a,
.caption a {
    color: #3b82f6 !important;
    text-decoration: none;
    border-bottom: 1px solid rgba(59,130,246,0.3);
    font-weight: 600;
}

.text-post-content a:hover,
.caption a:hover {
    border-bottom-color: #3b82f6;
    text-decoration: none;
}

/* Make sure links are visible on dark backgrounds */
.text-post-content a {
    text-shadow: 0 1px 1px rgba(0,0,0,0.3);
}
```

5. Alternative: Simple JavaScript solution (if template tags are complex)

If you prefer JavaScript, add this to your feed.html script section:

```javascript
// Make links clickable in post content
function makeLinksClickable() {
    document.querySelectorAll('.caption, .text-post-content .short-text, .text-post-content .full-text').forEach(element => {
        let html = element.innerHTML;
        // Match URLs
        const urlRegex = /(https?:\/\/[^\s]+|www\.[^\s]+|[a-zA-Z0-9.-]+\.[a-zA-Z]{2,}(?:/[^\s]*)?)/g;
        html = html.replace(urlRegex, function(url) {
            let href = url;
            if (!href.startsWith('http')) {
                href = 'https://' + href;
            }
            let displayUrl = url.length > 50 ? url.substring(0, 40) + '...' : url;
            return `<a href="${href}" target="_blank" rel="noopener noreferrer" style="color: #3b82f6; text-decoration: none; border-bottom: 1px solid rgba(59,130,246,0.3);">${displayUrl}</a>`;
        });
        if (element.innerHTML !== html) {
            element.innerHTML = html;
        }
    });
}

// Call after posts load
setTimeout(makeLinksClickable, 500);

// Also call when loading more posts
const observer = new MutationObserver(function(mutations) {
    makeLinksClickable();
});
observer.observe(document.getElementById('feed'), { childList: true, subtree: true });
```

6. For the "See more" button to work with links

Update your toggleTextContent function to re-apply linkify after expanding:

```javascript
function toggleTextContent(postId, event) {
    event.stopPropagation();
    const container = document.getElementById(`text-content-${postId}`);
    if (!container) return;
    const shortText = container.querySelector('.short-text');
    const fullText = container.querySelector('.full-text');
    const btn = container.querySelector('.see-more-btn');
    
    if (fullText && shortText && btn) {
        if (fullText.style.display === 'none' || fullText.style.display === '') {
            shortText.style.display = 'none';
            fullText.style.display = 'block';
            btn.textContent = 'See less';
            // Re-apply linkify to full text
            makeLinksClickable();
        } else {
            shortText.style.display = 'block';
            fullText.style.display = 'none';
            btn.textContent = 'See more';
        }
    }
}
```

✅ Summary

The template filter method is cleaner and works with server-side rendering. The JavaScript method is easier to implement if you want a quick fix without creating new files.

Both methods will:

· Detect any URL in post content
· Convert it to a clickable <a> tag
· Open links in a new tab
· Style links to be visible and look good

After adding this, any post containing a link will have it automatically clickable!
